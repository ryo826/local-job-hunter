Local Job Hunter セットアップガイド
=====================================

■ 必要なもの
  - Windows PC
  - インターネット接続

■ 手順1：Node.js をインストールする

  1. ブラウザで https://nodejs.org を開く
  2. 「LTS」と書いてある緑色のボタンをクリックしてダウンロードする
  3. ダウンロードしたファイルをダブルクリックしてインストーラーを起動する
  4. 「Next」を押していき、最後に「Install」を押す
  5. 完了したら「Finish」を押す

■ 手順2：ソースコードをダウンロードする

  1. ブラウザでこのプロジェクトのGitHubページを開く
  2. 緑色の「Code」ボタンをクリックする
  3. 「Download ZIP」をクリックしてZIPファイルをダウンロードする
  4. ダウンロードしたZIPファイルを右クリックして「すべて展開」を選ぶ
  5. 展開先を選んで「展開」を押す
     （例：デスクトップやドキュメントフォルダなど好きな場所でOK）

■ 手順3：コマンドプロンプトを開く

  1. キーボードの Windowsキー を押す
  2. 「cmd」と入力する
  3. 「コマンドプロンプト」が表示されるのでクリックして開く
  4. 黒い画面が表示されたら、展開したフォルダに移動する

     以下を入力してEnterキーを押す（パスは展開先に合わせて変更してください）：
     cd デスクトップ\local-job-hunter-main

     ※フォルダ名がわからない場合は、展開したフォルダを開いて
       アドレスバーのパスをコピーして「cd 」の後に貼り付けてください

■ 手順4：必要なファイルを自動ダウンロードする

  コマンドプロンプトで以下を1行ずつ入力し、それぞれEnterキーを押す：

  1. npm install
     → しばらく待つ（数分かかる場合があります）
     → 完了すると再び入力できる状態になります

  2. npx playwright install chromium
     → スクレイピングに使うブラウザが自動でダウンロードされます
     → これも数分かかる場合があります

■ 手順5：環境設定ファイルを作成する

  1. 展開したフォルダの中にある「.env.example」というファイルを見つける
     ※ファイルが見えない場合は、エクスプローラーの「表示」メニューから
       「隠しファイル」にチェックを入れてください
  2. このファイルをコピーして、同じフォルダ内に貼り付ける
  3. コピーしたファイルの名前を「.env」に変更する
     ※「拡張子を変更しますか？」と聞かれたら「はい」を押す
  4. 「.env」ファイルをメモ帳で開く
  5. 「GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here」の行を見つけて、
     「your_google_maps_api_key_here」の部分を自分のAPIキーに書き換える
  6. 保存して閉じる

  ※ Google Maps APIキーの取得方法：
    https://console.cloud.google.com/apis/credentials にアクセスし、
    Googleアカウントでログインして「APIキーを作成」から取得できます
  ※ Supabase URLとanon keyの行はそのままでOKです（変更不要）

■ 手順6：アプリをビルドして起動する

  コマンドプロンプトで以下を1行ずつ入力し、それぞれEnterキーを押す：

  1. npm run build
     → ビルドが完了するまで待つ

  2. npm run dev:electron
     → アプリが起動します

  ※ 2回目以降はビルド済みなので「npm run dev:electron」だけで起動できます

■ うまくいかないときは

  - 「'npm' は認識されていません」と表示される場合
    → Node.js のインストールが完了していません。手順1をやり直してください。
    → インストール後、コマンドプロンプトを一度閉じて開き直してください。

  - npm install で赤いエラーが大量に出る場合
    → Node.js のバージョンが古い可能性があります。
      「node -v」と入力して、v18以上であることを確認してください。
