Local Job Hunter セットアップガイド（Mac版）
=============================================

■ 必要なもの
  - Mac（macOS）
  - インターネット接続

■ 手順1：Node.js をインストールする

  1. ブラウザで https://nodejs.org を開く
  2. 「LTS」と書いてある緑色のボタンをクリックしてダウンロードする
     ※ Apple Silicon（M1/M2/M3/M4）の場合は「macOS Installer (.pkg)」を選択
  3. ダウンロードした .pkg ファイルをダブルクリックしてインストーラーを起動する
  4. 画面の指示に従って「続ける」→「インストール」を押す
  5. パスワードを求められたらMacのログインパスワードを入力する
  6. 完了したら「閉じる」を押す

■ 手順2：ソースコードをダウンロードする

  1. ブラウザで以下のURLを開く
     https://github.com/ryo826/local-job-hunter
  2. 緑色の「Code」ボタンをクリックする
  3. 「Download ZIP」をクリックしてZIPファイルをダウンロードする
  4. ダウンロードしたZIPファイルをダブルクリックすると自動で展開される
     （通常「ダウンロード」フォルダに展開されます）

■ 手順3：ターミナルを開く

  1. Finder を開く
  2. メニューバーの「移動」→「ユーティリティ」をクリックする
  3. 「ターミナル」をダブルクリックして開く
     ※ または Spotlight検索（Command + スペース）で「ターミナル」と入力してもOK
  4. 展開したフォルダに移動する

     以下を入力してEnterキーを押す（パスは展開先に合わせて変更してください）：
     cd ~/Downloads/local-job-hunter-main

     ※フォルダの場所がわからない場合は、ターミナルに「cd 」と入力した後、
       Finderからフォルダをターミナルにドラッグ＆ドロップしてください

■ 手順4：必要なファイルを自動ダウンロードする

  ターミナルで以下を1行ずつ入力し、それぞれEnterキーを押す：

  1. npm install
     → しばらく待つ（数分かかる場合があります）
     → 完了すると再び入力できる状態になります

  2. npx playwright install chromium
     → スクレイピングに使うブラウザが自動でダウンロードされます
     → これも数分かかる場合があります

■ 手順5：環境設定ファイルを作成する

  ターミナルで以下のコマンドを入力してEnterキーを押す：

  1. cp .env.example .env
     → 設定ファイルのひな形がコピーされます

  2. open -e .env
     → テキストエディットで .env ファイルが開きます
     ※ open -e で開けない場合は「nano .env」で編集できます

  3. 「GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here」の行を見つけて、
     「your_google_maps_api_key_here」の部分を自分のAPIキーに書き換える
  4. 保存して閉じる
     ※ nano の場合は Control + O → Enter → Control + X で保存して閉じます

  ※ Google Maps APIキーの取得方法：
    https://console.cloud.google.com/apis/credentials にアクセスし、
    Googleアカウントでログインして「APIキーを作成」から取得できます
  ※ Supabase URLとanon keyの行はそのままでOKです（変更不要）

■ 手順6：アプリをビルドして起動する

  ターミナルで以下を1行ずつ入力し、それぞれEnterキーを押す：

  1. npm run build
     → ビルドが完了するまで待つ

  2. npm run dev:electron
     → アプリが起動します

  ※ 2回目以降はビルド済みなので「npm run dev:electron」だけで起動できます

■ うまくいかないときは

  - 「command not found: npm」と表示される場合
    → Node.js のインストールが完了していません。手順1をやり直してください。
    → インストール後、ターミナルを一度閉じて開き直してください。

  - npm install で赤いエラーが大量に出る場合
    → Node.js のバージョンが古い可能性があります。
      「node -v」と入力して、v18以上であることを確認してください。

  - 「permission denied」と表示される場合
    → コマンドの先頭に「sudo 」を付けて再実行してください。
      例：sudo npm install
    → パスワードを求められたらMacのログインパスワードを入力してください。
